### `aria2` unlocked

Built from the latest source code of [aria2/aria2](https://github.com/aria2/aria2), this repository removes the maximum download connection limit and automatically compiles the patched binary for **Windows x64**.

#### Description

* **[Auto-Sync & Build]** Synchronizes with the source repository daily; automatically builds a new binary if updates are detected.
    
* **[Connection Limit Removed]** Removes the default 16-connection limit for `max-connection-per-server` during compilation.
    
* **[Windows x64 Ready]** The compiled `aria2c.exe` is portable and ready to use immediately upon download.
    

#### Download

Download the latest build of `aria2c.exe` from [Releases](../../releases/latest).

> #### [English](#aria2-unlocked) | **简体中文**

### `aria2` 解锁版

本仓库基于 [aria2/aria2](https://github.com/aria2/aria2) 项目最新源码，移除了最大下载连接数限制，并自动为 **Windows x64** 平台编译移除限制后的二进制程序。

#### 介绍

* **【自动同步与构建】** 每日同步源仓库代码，若检测到更新，则自动编译新版二进制程序。
* **【移除连接数限制】** 编译过程中，去除最大连接数 `max-connection-per-server` 默认为 `16` 的限制。
* **【Windows x64 开箱即用】** 编译得到的 `aria2c.exe` 文件下载即可直接使用。

#### 下载

可在 [Releases](../../releases/latest) 中下载最新构建的 `aria2c.exe`。
